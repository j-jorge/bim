Bim! Privacy Policy

Bim! is a game developed by Julien Jorge, but it does not matter. It is free software with no ad, no tracker, no trick for the end user.

I do not collect personal data but I do collect anonymous, non-identifying data. For example (non-exhaustive):

  • the language in which your device is configured, so I know which translations I should work on,
  • which buttons are clicked, so I know if something is useless in the user interface,
  • which errors happen, so I know if there is an issue I should address.

These data are collected using PostHog and stored in Europe.

Note that since this is an online game, I must also necessarily get your device's IP address. At the time of this writing this address is stored in a single place, in the logs of the web server when the game fetches the server list at launch time.
